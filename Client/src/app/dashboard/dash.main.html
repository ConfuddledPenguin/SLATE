Welcome to slate

SLATE is a tool designed to help you kee track of your progress within modules. It displays both current and old data to keep you informed.

Please note that while the SLATE system aims to be as accurate as possible poor data being fed into the system while result in poor data being presented to users.

New users are encouraged to go to your profile and set your learning goals. SLATE uses this to generate more meaningful information for you. Your profile can be reached by selected user options in the top right hand corner

To get more accurate feedback, each module also presents the option the set goals. Modules can be viewed by using the navigation on the left.

Lastly an overview of your warnings are displayed below, for your stressful viewing pleasure :)
